# Brand Naming Candidates — Multistreaming Desktop App

Context: fork of OBS Studio (CEF+Svelte frontend), one encode fanned out to many platforms (Twitch/YouTube/Kick, later Facebook). Name must avoid "OBS", avoid bare-descriptive terms ("Multistream(er)"), and work as the OAuth consent-screen app name. Direct competitors: Restream, StreamYard, Streamlabs, OneStream, Castr.

**Scope note:** the checks below are quick WebSearch collision scans only — not a substitute for real USPTO/TESS search, domain registration checks, and social-handle checks. Treat every "looks clean" verdict as *no obvious collision found*, not *cleared*. That clearance work is the user's job before committing to a name.

## 3. Ranked top 3

| Rank | Name | Why |
|------|------|-----|
| **#1** | **Braidcast** | Cleanest search result of any candidate, strongest metaphor fit (braiding = combining many strands from one source, exactly what fan-out multistreaming does), fanciful/suggestive rather than descriptive (better trademark strength than "Splitbeam" or the bare "Multistream(er)" the constraints rule out), easy to say/spell, plausible `.com`/`.gg` domain and social handle. |
| #2 | **Splitbeam** | Also came back clean, and the visual metaphor is instantly graspable — but "split" + "beam" are both literal, everyday words describing the mechanism fairly directly, which is legally weaker (more descriptive, less distinctive) than a name like Braidcast and slightly closer to the "avoid bare-descriptive" instruction than is ideal. |
| #3 | **Wavora** | The only fully-coined option with a genuinely clean signal (one dead, unrelated-class trademark from 2013), giving it the strongest fanciful-mark legal footing of all 15 — but it has the weakest conceptual tie to "one source, many destinations," so it would lean entirely on visual identity/marketing to carry meaning, unlike Braidcast which does some of that work for free. |

**Recommendation: Braidcast** — best balance of a clean collision scan, a distinctive (non-descriptive) trademark profile, and a metaphor that actually explains the product.

Real clearance still owed before committing: live USPTO/TESS search, EU/UK equivalents if relevant, `.com`/`.gg`/`.tv` domain availability, and handle availability on the platforms this app integrates with (Twitch/YouTube/Kick/X) — none of that was done here.

## 4. Braidcast — independent verification (Playwright)

Follow-up pass done live in-browser (2026-07-04) against the successor to TESS, plus a domain check and a fresh product-collision search. This is a **preliminary signal, not a legal clearance** — no attorney, no full international/common-law search, no exhaustive class review.

**USPTO trademark search** (`tmsearch.uspto.gov`, the "Trademark Search" system that replaced classic TESS):
- Exact query `Braidcast` (Wordmark search, all statuses: Live/Registered/Pending/Dead/Cancelled/Abandoned, all classes) → **"No results found."** Screenshot: `braidcast-uspto-results.png`.
- Broadened query `braid cast` (two words) → 2,147 results, but this is the search engine's fuzzy/token matching independently on "BRAID" and "CAST" across unrelated goods (adhesives, perfumes, fishing line, etc.) — spot-checked the result cards directly and found **no mark whose wordmark text is actually "BRAIDCAST" or "BRAID CAST"** anywhere in the list. Screenshot: `braidcast-uspto-braidcast-two-word.png`.
- Read as: no live or dead U.S. wordmark registration/application for "Braidcast" as such, in any class — including the relevant IC 009 (software), 038 (telecommunications/broadcast), 042 (SaaS) buckets.

**Domain:**
- `braidcast.com` — DNS does not resolve (`ERR_NAME_NOT_RESOLVED` on direct navigation) and `whois.com/whois/braidcast.com` explicitly reports **"Looks like this domain has not been registered yet"** (offered for registration at $13.48). Screenshot: `braidcast-whois.png`.
- `.tv` / `.gg` / `.io` / `.app` variants not checked in this pass (time-boxed to `.com`) — quick registrar check still owed if `.com` is contested later (it isn't).

**Existing products / collisions:**
- WebSearch for `"Braidcast" streaming app OR software` returned no matching product — only unrelated near-miss results (Wirecast, Streamcast, BoxCast, AzuraCast, Streamlabs).
- WebSearch for `"Braidcast" trademark` surfaced one real usage: **"Braidcast, episode 01: Seph Gentle (Talking about CRDT's)"** (Hacker News, [news.ycombinator.com/item?id=31437460](https://news.ycombinator.com/item?id=31437460)) — an episode-title in a talk/podcast series associated with the *Braid* protocol/sync community (braid.org, unrelated to broadcast/streaming software). Not a product name, not a company, not a registered mark, and outside our category (live-streaming desktop software) — low relevance, worth knowing about only because it's the one organic "Braidcast" string on the open web.
- No other company, app, or software product using "Braidcast" was found.

**Verdict: CLEAR (proceed to formal clearance).** No USPTO hit, no live domain, no product collision in or near the streaming-software space — the strongest signal of the three candidates re-checked so far. This is still a preliminary automated signal; get a real attorney-run USPTO/TESS + common-law + international search before filing or committing spend to the name.
